# 3D Batman Detection using onshape

This is a university project where I designed a 3D Batman shape and trained a machine learning model to recognize it using Google's **onshape**.

## 🛠️ How it Works
1. **3D Design:** Created a symmetrical 3D model of the Batman logo.
2. **Data Collection:** Took multiple photos of the 3D shape from different angles.
3. **Training:** Uploaded the photos to onshape to train the model.
4. **Testing:** Tested the model to make sure it recognizes the Batman shape correctly.

## 💻 Technologies Used
* Google onshape
* 3D Modeling Software
* GitHub
